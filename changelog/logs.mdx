## October, 2024

### October 11, 2024

- **Chain Support Expansion:** Added support for Movement Devnet, Lumia, Soneium, Duckchain Mainnet
- **Chain Update:** Updated Gravity Alpha to Gravity, update Plume and Story Network's RPC
- **BTC Connect:** Add support for XLayer and Elastos

## August, 2024

### August 23, 2024

- **New Feature:** *Passkey for Smart Account* - A new method to sign transactions without the need for a private key.
- **Smart Account Update:** Introduced the *Coinbase Smart Account*, fully compatible with Passkey from the start.
- **Chain Support Expansion:** Added support for Gravity Alpha, Story Network, DODOChain, Duckchain, and Plume.
- **Biconomy V2 Update:** Expanded chain support with the addition of Sei.
- **Chain Removal:** Discontinued support for Heco and GMNetwork Testnet.

### August 9, 2024

- **Bug Fix:** Resolved intermittent issues with Solana transaction failures.
- **Performance Improvement:** Optimized loading speed for Solana transaction history.
- **Bug Fix:** Corrected the issue preventing Solana from adding manually tracked tokens.

### August 5, 2024

- **Bug Fix:** Resolved an issue with GitHub login where canceling would not return the user to the login page.
- **Bug Fix:** Addressed issues with Microsoft login.

### August 1, 2024

- **Stability Improvement:** Enhanced the stability of Node RPC.
- **Bug Fix:** Fixed issues related to gas price calculations.
